# 图书管理系统（Flask + Next.js 前后端分离实训项目）
## 一、项目介绍
本项目为个人独立实训前后端分离图书管理系统，实现管理员账号登录、图书列表查询、新增图书功能；后端使用Flask提供3组RESTful接口，前端基于Next.js App Router实现3个独立页面路由，搭配Tailwind CSS完成页面样式。
## 二、技术栈
### 后端
Python 3.14、Flask、Flask-CORS、SQLite3
### 前端
Next.js 14、TypeScript、Tailwind CSS、Axios
