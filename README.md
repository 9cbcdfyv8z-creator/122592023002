# 图书管理系统（Flask + Next.js 前后端分离实训）
## 一、项目简介
个人独立完成前后端分离图书管理系统，实现管理员登录、图书列表查询、新增图书功能。后端3组RESTful接口，前端3个独立路由页面，配套完整实训文档、AI开发记录、线上部署链接。
## 二、技术栈
### 后端
Python3.14、Flask、Flask-CORS、SQLite3
### 前端
Next.js14 App Router、TypeScript、Axios、Tailwind CSS
